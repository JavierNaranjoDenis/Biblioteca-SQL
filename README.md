#Sistema de Gestión Bibliotecaria/
# Descripción

Proyecto desarrollado en MySQL para la administración de una biblioteca.

Permite gestionar usuarios, libros, autores y préstamos mediante una base de datos relacional.

#Objetivos

- Aplicar normalización de bases de datos.
- Implementar relaciones entre entidades.
- Realizar consultas SQL de nivel básico e intermedio.
- Generar reportes operativos.

#Tecnologías

- MySQL
- SQL

#Funcionalidades

- Registro de usuarios.
- Registro de libros.
- Registro de autores.
- Control de préstamos.
- Control de devoluciones.
- Reportes estadísticos.
